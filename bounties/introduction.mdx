---
title: "Bounties"
description: "Bounties are active issues on repositories that have been imported on the Gitpaid platform."
---

Bounties on GitPaid are issues on repositories that offer rewards for successful completion. These can range from fixing bugs and implementing new features to writing documentation and designing graphics. By completing bounties, developers earn rewards while actively contributing to open-source projects.
