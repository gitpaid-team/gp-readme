### Explore More Projects

<ul>
  <li>
    <strong>Search All of GitHub:</strong> Use our tool to search for any GitHub
    project that you want to support.
  </li>

    <strong>Invite Repositories to Join GitPaid:</strong> If the project isn’t

    welcome bonus for signing up, making your support even more impactful!
  </li>
</ul>
