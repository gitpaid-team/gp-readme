---
title: "Overview"
description: "Issues are active problems or challenges on Github Repositories that have been imported into the Gitpaid platform."
---

Issues on GitPaid are offer rewards for successful completion. These can range from fixing bugs and implementing new features to writing documentation and designing graphics. By driving an issue to its completion, developers earn rewards while actively contributing to open-source projects.
