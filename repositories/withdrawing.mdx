## Withdrawing from a repository

You can withdraw funds from a repository directly via the repository card within your _Profile_. Withdrawing is only possible under the following circumstances.

<ul>
  <li>
    The amount you would like to withdraw is available within the Repository.
  </li>
  <li>The Unlock Time has elapsed.</li>
</ul>
